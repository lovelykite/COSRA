# COSRA (Colored Object Sorting Robotic Arm)

- Design a bowden-cable driven 3-axis manipulator
- Colored object recognition & sorting based on OpenCV
- Generate cartesian trajectory with LSPB
- Term Project during RT750 Robot System Design, DGIST
